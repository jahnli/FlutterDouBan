# Flutter -  Douban

## 介绍
基于Flutter，使用豆瓣真实App数据，尽可能还原精美的豆瓣App，功能会持续添加，以此来深入了解Flutter。
如您有更好的意见或优化，请提出Issue。

目前还在Coding阶段，在逐步增加功能页面，学习Flutter阶段，后期会考虑优化，如有专业大佬请指教。

> * git clone https://github.com/jahnli/FlutterDouBan
> * cd FlutterDouBan
> * flutter run
